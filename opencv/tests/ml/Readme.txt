To run mltest you must specify config_filename (-f) and data_path (-d).
config_filename  -  full name of file OpenCV_SVN/tests/ml/testdata/validation.xml;
data_path  -  full path to dataset files - OpenCV_SVN/tests/ml/testdata.

For example
mltest.exe -f D:/Program Files/tests/ml/testdata/validation.xml -d D:/Program Files/OpenCV/tests/ml/testdata
